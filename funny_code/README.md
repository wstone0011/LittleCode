## 2.欧派曲线
```
x = np.arange(1,0,-0.001)
y = (-4 * x * np.log(x) + np.exp(-(40*(x - 1 / np.e)) ** 4) / 10) /2
plt.figure(figsize=(2,5))
plt.plot(y, x, 'r-', linewidth=2)
plt.show()
```
